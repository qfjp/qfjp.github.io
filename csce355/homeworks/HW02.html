1.
Find a counterexample to the statement, “All odd natural numbers are prime.” What is the least counterexample (NOTE: 1 is NOT prime)?
2.

Here are some standard definitions of set operations. I’m using the symbol “\(:=\)” to mean “equals by definition.” For any sets \(A\) and \(B\),

Union:
\(A\cup B := \{ x \st x\in A \myor x\in B \}\).
Intersection:
\(A\cap B := \{ x \st x\in A \myand x\in B \}\).
Difference (Relative Complement):
\(A-B := \{ x \st x\in A \myand x\notin B \}\).
Symmetric Difference:
\(A\symdiff B = (A-B)\cup (B-A)\).

For each of the four operations above, redraw the Venn diagram below, shading the region corresponding to the result of the operation.

3.
Illustrate via a venn diagram the following identity for any sets \(A\) and \(B\). \[ A\symdiff B = (A\cup B) - (A\cap B). \]
4.
Prove the identity in the last problem directly for any sets \(A\) and \(B\).
